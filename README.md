## ✅ 수정사항 올리기
### 👣 1. 변경사항 확인
```bash
git status
```
### 👣 2. Git에 추가 + 커밋
```bash
git add .
git commit -m "Fix: 학생 QnA 페이지 UI 개선"
git rebase --continue
```
### 👣 3. GitHub에 푸시!
```bash
git push origin main
```

## ✅ 수정사항 받아오는 방법
### 👣 1단계: 내 `test` 폴더로 이동
```bash
cd ~/test
```
### 👣 2단계: 현재 상태 확인 (혹시 내 수정사항이 남아있을 수도 있으니까)
```bash
git status
```
### 👣 3단계: GitHub에서 팀원 변경사항 가져오기
```bash
git pull origin main
```
